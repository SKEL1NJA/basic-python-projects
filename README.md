# Python Foundations: Core Projects Portfolio

A collection of four Python applications built to master foundational software engineering concepts, ranging from command-line interfaces to Object-Oriented Programming (OOP) and data persistence.

## 🚀 Projects Overview

This repository contains four distinct projects, each focusing on a specific set of Python skills:

### 1. Simple Banking System
A command-line application that simulates a basic bank account.
* **Key Features:** Deposit, withdraw, and check balance with robust error handling to prevent invalid inputs.
* **Skills Applied:** Object-Oriented Programming (Classes, Encapsulation), State Management, and File I/O (JSON).

### 2. Contact Manager CLI
A functional terminal-based application to manage a list of personal contacts.
* **Key Features:** Add, view, and delete contacts. Data is automatically saved and loaded across sessions.
* **Skills Applied:** Functional Programming, Dictionary/List manipulation, and JSON data persistence.

### 3. E-commerce Sales Data Analyzer
A data processing script designed to extract insights from raw sales data.
* **Key Features:** Filters completed orders, sorts by revenue, and calculates total sales.
* **Skills Applied:** List/Dict Comprehensions, Lambda functions, and Argument Unpacking (`*args`).

### 4. Library Management System
An advanced OOP application modeling the interactions between library staff, members, and books.
* **Key Features:** Librarians can add books to the inventory, while members can borrow and return books. Prevents duplicate checkouts.
* **Skills Applied:** Advanced OOP (Class Inheritance), Sets, and multi-object interactions.

## 🛠️ Technologies & Concepts Used
* **Language:** Python 3.x
* **Core Libraries:** `json`, `pprint`
* **Concepts:** OOP, Inheritance, File I/O, Error Handling (`try/except`), Comprehensions, Lambdas, CLI State Management.

## 💻 How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/SKEL1NJA/basic-python-projects.git
   ```

2. Navigate into the project folder of your choice:
   ```bash
   cd "Library management system"
   ```

3. Run the main Python script:
   ```bash
   python main.py
   ```
